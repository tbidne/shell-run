Similar to tryCommand except we attempt to stream the commands' output to a ConsoleRegion instead of the usual swallowing.

Since: 0.1

tryCommandStreamNoRegion :: (HasCompletedCmds env, HasLogging env, MonadReader env m, Mutable m, MonadUnliftIO m, RegionLogger m, Region m ~ ConsoleRegion, Timing m) => Command -> m (Maybe Stderr) Source #

We stream the commands' output like tryCommandStreamRegion except we do not create a console region. This function is intended for when we want to send command logs to a file, but do not want to stream them to the console.
