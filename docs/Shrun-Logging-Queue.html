Description

Provides the LogTextQueue type and associated functions. This is intended for concurrently writing logs to a file.

LogText is a textual representation of a given Log. No coloring is included, but we include the prefix (e.g. Warn) along with a timestamp.

Examples

Expand
formatFileLog $ MkLog Nothing "Running time: 2 seconds" Info Set LogBoth
"[2022-02-23 20:58:04.231933782 UTC] [Info] Running time: 2 secondsn"

formatFileLog $ MkLog (Just "cmd") "cmd: command not found" Error Set LogBoth
"[2022-02-23 20:58:04.231933782 UTC] [Error] [cmd] cmd: command not foundn"
