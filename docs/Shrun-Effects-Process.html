tryCmdStream :: Command -> m (Maybe Stderr) Source #

We stream the commands' output like tryStreamRegion except we do not create a console region. This function is intended for when we want to send command logs to a file, but do not want to stream them to the console.

Since: 0.5

tryCmdStreamRegion :: Command -> m (Maybe Stderr) Source #

Similar to tryCmd except we attempt to stream the commands' output to a Region.
